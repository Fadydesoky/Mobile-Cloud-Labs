# Frontend - Mobile Cloud Dashboard

It is designed to complement the cloud backend by providing a minimal user interface for interaction.

A simple frontend interface built to interact with the backend API.

It simulates a mobile client that communicates with cloud services through HTTP requests.

---

## Purpose

- Provide a visual way to test API endpoints  
- Demonstrate client-server interaction  
- Represent the mobile side of the system  

---

## Features

- Response time display  
- Data retrieval  
- Health status check
